# Jeff Ross website - The Roast Master General

http://roastmastergeneral.com/

## Running locally

- Clone the repo:
```
git clone https://github.com/claudioccm/jeffross.git jeffross
```

- Go to the project's directory:
```
cd jeffross
```

- Install the dependencies:
```
npm install
```

Run the server:
```
gulp
```

## Deployment

```
gulp deploy
```
